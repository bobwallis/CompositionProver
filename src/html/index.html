This page contains a simple web interface for Richard Smith's GSiril touch proving program.

It contains:

  1. a code editor that supports syntax highlighting;
  2. a version of the GSiril program that has been compiled to Javascript so that it can run in a web browser;
  3. some code to join these two things together.

The source code is made available for interested people, and released under the GPL.
